# 🌦️ Prodigy Weather App

A full-stack weather application that fetches real-time weather data using the OpenWeatherMap API. Users can search weather by city name or use their current location to view live weather conditions.

---

### Frontend

### Backend API
(https://prodigy-wd-05-sltx.onrender.com)

---

## 📌 Features

- 🌍 Search weather by city
- 📍 Get weather using current location
- 🌡️ Real-time temperature and weather conditions
- 💨 Wind speed, humidity, pressure, visibility
- 🕘 Recent search history
- 📱 Responsive modern UI
- ⚡ Fast API integration
- 🔗 Full-stack deployment

---
